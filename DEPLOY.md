# 🚀 מדריך העלאת "מי ומה" לדומיין + הפניות 301 מהאתר הישן (Site123)

## שלב 1 — העלאת האתר החדש ל-Vercel (חינם)

1. היכנסי ל-https://vercel.com והירשמי (הכי קל עם חשבון GitHub).
2. העלי את תיקיית הפרויקט ל-GitHub (ריפוזיטורי חדש בשם `mevema`).
   - אם אין לך GitHub: אפשר גם דרך שורת פקודה עם `npx vercel` מתוך תיקיית הפרויקט.
3. ב-Vercel: **Add New → Project → Import** את הריפוזיטורי `mevema`.
4. Vercel מזהה אוטומטית שזה פרויקט Vite. לפני Deploy, פתחי **Environment Variables** והוסיפי:
   - `VITE_SUPABASE_URL` = הכתובת של פרויקט ה-Supabase שלך
   - `VITE_SUPABASE_ANON_KEY` = מפתח ה-anon של Supabase
5. לחצי **Deploy**. תוך דקה תקבלי כתובת זמנית כמו `mevema.vercel.app` — בדקי שהאתר עובד.

## שלב 2 — חיבור הדומיין שלך

1. ב-Vercel: **Settings → Domains → Add** והקלידי את הדומיין (למשל `mevema.co.il`).
2. Vercel יציג לך אילו רשומות DNS להגדיר. היכנסי לחברה שבה רשום הדומיין (למשל: איזיפי, GoDaddy, דומיינר, טוליפ) ועדכני:
   - רשומת **A** עבור `@` → `76.76.21.21`
   - רשומת **CNAME** עבור `www` → `cname.vercel-dns.com`
3. ⚠️ **חשוב:** אם הדומיין מחובר כרגע ל-Site123 — עדכון הרשומות האלה **מנתק אותו מ-Site123 ומחבר אותו לאתר החדש**. זה בדיוק מה שאנחנו רוצים.
4. ההגדרות נקלטות בדרך כלל תוך שעה (יכול לקחת עד 24 שעות). Vercel מנפיק תעודת SSL (https) אוטומטית וחינם.

## שלב 3 — הפניות 301 מהאתר הישן

יש שני מצבים, תבדקי איזה מהם נכון אצלך:

### מצב א׳: האתר הישן ישב על הדומיין שלך (למשל mevema.co.il דרך Site123)
זה המצב הפשוט: ברגע שהדומיין מצביע ל-Vercel (שלב 2), כל מי שמגיע לכתובות הישנות מגיע אוטומטית לאתר החדש.
כדי שגם עמודים פנימיים ישנים (כמו `/about` או `/contact`) יופנו יפה לעמוד הבית עם קוד 301 — כבר הכנתי לך את הקובץ `vercel.json` בפרויקט. תוכלי להוסיף שם שורות לכל כתובת ישנה שהייתה באתר:

```json
{ "source": "/כתובת-ישנה", "destination": "/", "permanent": true }
```

("permanent": true = הפניית 301 קבועה, ששומרת על הדירוג בגוגל.)

### מצב ב׳: האתר הישן ישב על כתובת של Site123 (למשל michal.site123.me)
ב-Site123 אי אפשר להגדיר הפניות 301 אמיתיות. מה שכן אפשר:
1. **בתוכנית בתשלום של Site123** עם דומיין: לנתק את הדומיין משם ולחבר אותו ל-Vercel (מצב א׳).
2. אם האתר הישן נשאר על כתובת site123.me: היכנסי לעורך של Site123, מחקי את התוכן והשאירי עמוד אחד עם הודעה + קישור: **"עברנו לכתובת חדשה! mevema.co.il"**. אחרי חודש-חודשיים אפשר לסגור את חשבון Site123.
3. עדכני את הקישור בכל מקום שפרסמת אותו (גוגל לעסק שלי, פייסבוק, אינסטגרם, חתימת מייל).

## שלב 4 — גוגל (מומלץ, 10 דקות)

1. היכנסי ל-https://search.google.com/search-console והוסיפי את הדומיין החדש.
2. אם האתר הישן היה רשום שם — השתמשי בכלי **Change of Address** כדי להודיע לגוגל על המעבר.
3. כך הדירוג עובר לאתר החדש תוך מספר שבועות.
